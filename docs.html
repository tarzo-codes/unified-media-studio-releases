Overview

Unified Media Studio is a professional, offline-first media processing suite built using Qt6 and C++. It aims to unify powerful backend libraries like FFmpeg, auto-editor, and OpenAI Whisper into a responsive, local desktop application with zero cloud tracking or telemetry.

Whether you need to edit audio waveforms, transcribe podcast episodes, convert media pipelines in bulk, or trim silences instantly, Unified Media Studio handles it natively on your processor or GPU.

Offline First: Unified Media Studio processes all computations entirely on your local machine. No API keys, server tokens, or network requests are ever dispatched during processing.

Installation Guide

Unified Media Studio is available as packaged builds for both Windows and popular Linux distributions. Please read the instructions below to configure the app on your environment.

Windows

Select either the setup installer or portable zip from the downloads section:

  • Setup Installer (Recommended): Runs a standard installation wizard. Offers an automated option to download and install core dependencies like Python and FFmpeg on first launch.
  • Portable Zip: Self-contained directory structure. Extract the directory anywhere and double-click windows/Launch.bat to launch the studio.

Linux Distributions

Ensure that standard packages (like Python and FFmpeg) are pre-installed via your system's package manager before deploying the application.

Distribution Prerequisites Command Installation Command
Ubuntu / Debian sudo apt install ffmpeg python3 python3-pip sudo dpkg -i UnifiedMediaStudio_v1.1.5-alpha-Linux-x64.deb
Fedora / RedHat sudo dnf install ffmpeg python3 sudo dnf install UnifiedMediaStudio_v1.1.5-alpha-Linux-x64.rpm
Arch Linux sudo pacman -S ffmpeg python sudo pacman -U UnifiedMediaStudio_v1.1.5-alpha-Arch-x64.pkg.tar.zst

✂️ Silence Remover

The Silence Remover parses your video and audio tracks, analyzes decibel levels, and trims silent parts with absolute precision.

Configurability:

  • Silence Threshold: Set the decibel volume range to classify as silence (Default: -35.00 dB).
  • Min Silence Duration: The minimum length of time in seconds that a quiet section must last to be cut.
  • Padding: Keeps a short pad before and after cut points to ensure vocal inflections aren't abruptly trimmed.

Export Modes:

  1. Video + XML: Outputs a compiled trimmed media file along with an Edit Decision List (EDL) XML. Perfect for import into Resolve or Premiere.
  2. Video Only: Instantly encodes and outputs only the trimmed video file.
  3. XML Only: Skips encoding entirely and exports only the cut-point XML file (completed in seconds).

🎙️ AI Transcriber

Offline audio transcription powered by OpenAI Whisper.

  • Whisper Models: Choose weights from tiny (super fast, low memory) up to large-v3 (industry-standard accuracy).
  • Device Selector: Choose CPU or CUDA-accelerated GPU for transcription. GPU execution requires Nvidia CUDA packages.
  • Subtitle Export: Auto-generates SubRip subtitle tracks (.srt) and structured JSON transcript objects.
  • Ollama Integration: Chains Whisper transcripts into a local Ollama LLM to summarize key chapters, create podcast titles, or bullet-point discussions natively.

🎬 Media Converter

Quickly convert media files using high-fidelity conversion profiles or configure custom overrides.

  • DNxHD — Editing Master: Renders lossless-quality video files with PCM audio, optimized for editing in Premiere or Resolve without CPU decoding strain.
  • MP4 H.264 / H.265: High-efficiency video encoders. Great for mobile uploads or storage compression.
  • Audio Presets: Export uncompressed 16-bit stereo WAVs or standard 320 kbps MP3 files.
  • Replace Audio Track: Strip existing video audio and multiplex a new audio file directly onto the timeline without re-encoding the video track.

🔊 Audio Editor

Features an interactive filter chain and DAW waveform analyzer.

  • Waveform Noise Profile Capture: Select a noise-only region on the visual waveform using your mouse cursor. Click "Capture Profile" to use that sample as a noise filter matrix coefficient.
  • Filters: Real-time control for Gain, Denoise strength, High/Low Pass Equalization, Compression thresholds, Limiter ceiling, and Reverb space.
  • Vocal Presets: Predefined mastering chains like Voice Clarity and Podcast Heavy Noise Reduction.

📋 Bulk Pipeline

The Bulk Processing Pipeline lets you orchestrate complex media flows on entire directories of files sequentially.

How to build a pipeline:

  1. Drag processing blocks (Silence Removal, Audio Enhance, Convert, Transcribe) from the tool list into the pipeline queue.
  2. Reorder steps visually.
  3. Click the settings gear icon on each step to configure independent parameters for that module.
  4. Click "Execute Pipeline" to run the queued operations batch-style across the staged files list.

Troubleshooting Guide

Review these solutions for common setup issues, particularly when deploying inside WSL or container systems.

1. Missing Fonts / Blank UI Text (Tofu Squares)

If labels inside the application display as empty boxes (□□□□), your Linux instance lacks a default font package. Run the corresponding command to install DejaVu glyph layouts:

# Arch Linux:
sudo pacman -S --needed --noconfirm ttf-dejavu fontconfig

# Ubuntu/Debian:
sudo apt-get install -y fonts-dejavu fontconfig

# Fedora:
sudo dnf install -y dejavu-sans-fonts fontconfig

2. Wayland/WSLg Window Fails to Render

On some Windows hardware, WSLg's GPU Vulkan/OpenGL translation wrapper will conflict with dynamic Qt drivers, causing the app window to crash or remain hidden on startup.

Solution: Force software-only graphics rendering by setting the environment variable:

LIBGL_ALWAYS_SOFTWARE=1 unified-media-studio

3. PipeWire Config Warnings

Warnings like pw.conf | can't load config client.conf are standard driver messages from WSL's audio stack. They do not affect media processing functionality and can be safely ignored.
