# Unified Media Studio — Stable Releases & Public Web Portal

This repository contains the official, stable production releases of **Unified Media Studio**, along with the interactive web portal interface.

---

## 🌐 Live Web Portal
The root of this repository contains the files for the interactive web portal. Users can:
* Test-run the application features (Silence Remover, AI Transcriber, Media Converter, Audio Editor) using the **Live Browser Simulator**.
* Download native installers and portable packages for Windows and Linux.
* Review key installation guides and troubleshooting steps.

To launch the portal locally, simply open [index.html](index.html) in any modern web browser.

---

## 📦 Stable Releases
All stable build assets are stored in the `/downloads` directory:
* **Windows Setup**: `downloads/UnifiedMediaStudio_v1.1.5-alpha-Setup-Windows.exe`
* **Windows Portable**: `downloads/UnifiedMediaStudio_v1.1.5-alpha-Portable.zip`
* **Ubuntu/Debian (.deb)**: `downloads/UnifiedMediaStudio_v1.1.5-alpha-Linux-x64.deb`
* **Fedora/RedHat (.rpm)**: `downloads/UnifiedMediaStudio_v1.1.5-alpha-Linux-x64.rpm`
* **Arch Linux (.tar.zst)**: `downloads/UnifiedMediaStudio_v1.1.5-alpha-Arch-x64.pkg.tar.zst`
